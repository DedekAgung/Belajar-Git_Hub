Kali ini saya mempraktekan bagaimana cara menggunakan github.
karena tidak banyak yang harus saya ringkas,maka saya catat poin penting yang saya dapat saja.


-Repository name tidak boleh menggunakan space,digati denga (_)
-Jika dipublic semua orang bisa mengakses repo kita, jika kita pilih private max 3 orang yang bisa akses dengan syyarat kita menerimanya,bisa lebih jika kita berlangganan
-readme akan menjadi file yang muncul pertama kali saat kita membuka repo.
-untuk membuat foler harus menambahkan / pada akhir nama file.
-untuk melihat commit dalam bentuk branch ke insight, lalu ke network ( hanya bisa dibuka jika public/private namun berlangganan).
